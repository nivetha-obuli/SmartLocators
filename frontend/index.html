📋 How to Get Raw HTML of Any Web Page
01
View Page Source Easiest

Open the page, press Ctrl+U (Win) or ⌘+U (Mac).
New tab opens → Ctrl+A → Ctrl+C → paste below.

⚠️ Static source only — JS-rendered content won't appear.
02
DevTools Elements Tab Recommended

Right-click page → Inspect → right-click <html> tag
→ Copy → Copy outerHTML → paste below.

💡 Right-click any specific <form> for partial HTML.
03
DevTools Console Live DOM

Press F12 → Console tab → type command and press Enter:

copy(document.documentElement.outerHTML)
💡 Best for login-protected or JavaScript-heavy pages.
04
Save Page As Offline

Press Ctrl+S → Save as: Webpage, HTML Only
Open saved .html in VS Code → Select All → Copy → paste below.

⚠️ Choose "HTML Only" not "Complete" to avoid extra folders.
🚀 Quick Pick: Use Method 02 for most pages. Use Method 03 for React/Angular SPAs or pages that require login.
